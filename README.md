##This repository contains a machine learning–based spam detection system developed as part of Task-4 of the CODTECH Internship program. The objective of this project is to design and implement a predictive classification model using the Scikit-learn library to automatically identify whether a given text message is spam or ham (non-spam). The implementation begins with preparing a labeled dataset of text messages, which serves as the foundation for supervised learning. Since machine learning algorithms cannot directly interpret raw text, the textual data is transformed into numerical feature vectors using the CountVectorizer technique, which converts words into frequency-based representations.

After feature extraction, the dataset is divided into training and testing subsets to ensure proper evaluation of model performance. A Multinomial Naive Bayes classifier is then trained on the training data to learn word distribution patterns commonly associated with spam and legitimate messages. The trained model is subsequently tested on unseen data to evaluate its predictive capability. Performance metrics such as accuracy are calculated to measure how effectively the model classifies messages. Additionally, the model is tested with custom input samples to demonstrate real-time prediction functionality.

This project highlights key concepts in machine learning, including text preprocessing, feature engineering, supervised classification, model training, and evaluation. It demonstrates how probabilistic algorithms can be applied to real-world text classification problems and showcases the practical application of Python and Scikit-learn in building intelligent predictive systems.
